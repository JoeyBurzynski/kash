# KASH

#### Simple and efficient in-memory caching

## Installation

Using [`npm`](http://npmjs.org/):

``` sh
$ npm [-g] install kash
```

In [Node.js](http://nodejs.org/):

``` javascript
var kash = require("kash");
```
